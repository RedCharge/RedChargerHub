# EduPal - Flask MVP

A simple web application for course management and learning.

## Features
- User authentication (Login/Register)
- Course management
- Learning resources
- Quizzes
- Responsive design

## Setup

1. Install dependencies:
\\\\\\ash
pip install -r requirements.txt
\\\\\\

2. Run the application:
\\\\\\ash
python run.py
\\\\\\

3. Open http://localhost:5000 in your browser

## Default Routes
- /login - User login
- /register - User registration
- / - Dashboard with courses
- /add_course - Add new course
- /resources - Learning resources
- /quizzes - Quiz section

## Database
Uses SQLite by default. The database file will be created automatically.
